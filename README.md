Trabajo propuesto

Para el obligatorio realice un juego que consiste en una pelea entre el knight y hueteolt, 
donde cada uno puede: 

- atacar al otro
- eludir un ataque
- rendirse y abandonar la pelea

Manual

Movimientos Knight:

W  -  Atacar
Q  -  Eludir
E  -  Rendirse	  

Movimientos Hueteolt:

O  -  Atacar
I  -  Eludir
P  -  Rendirse	

Teclas adicionales:

R  -  Reiniciar juego
L  -  Prender luz
M  -  Activar malla
Z  -  Activa z buffer
B  -  Activa cullface
